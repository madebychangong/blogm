# 블로그 원고 자동 최적화 시스템 사용 가이드

## 📋 개요

네이버 블로그 원고를 **SEO 최적화** 및 **C랭크 기준**에 맞춰 자동으로 수정하는 시스템입니다.

### 주요 기능

✅ **금칙어 자동 치환** - `금칙어 수정사항 모음.txt` 파일의 금칙어를 자동으로 대체어로 변경
✅ **SEO 최적화** - 키워드 반복(5-8회), 해시태그(8-10개) 자동 생성
✅ **AI 느낌 제거** - 반복적인 AI 문장 패턴을 다양한 표현으로 변경
✅ **네이버 C랭크 기준 체크** - 글자수, 키워드 밀도, 해시태그 개수 등을 분석하고 점수화
✅ **제목 자동 생성** - SEO 최적화된 제목 자동 생성 (15-40자)

## 🚀 빠른 시작

### 1. 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 엑셀 파일 준비

`작업 의뢰용 데이터.xlsx` 파일에 다음 컬럼이 있어야 합니다:
- **키워드**: 메인 키워드 (예: "팔꿈치 쿠션 보호대")
- **브랜드**: 브랜드명 (예: "초록입환")
- **원고**: 블로그 원고 내용
- **제목**: 블로그 제목 (선택사항 - 없으면 자동 생성)

### 3. 금칙어 파일 준비

`금칙어 수정사항 모음.txt` 파일 형식:
```
금칙어
대체어

금칙어2
대체어2

...
```

### 4. 최적화 실행

```bash
python3 blog_optimizer.py
```

## 🎯 네이버 C랭크 기준

### 점수 기준

| 등급 | 점수 | 설명 |
|------|------|------|
| **S** | 90점 이상 | 매우 우수 - 높은 노출 기대 |
| **A** | 80-89점 | 우수 - 양호한 노출 |
| **B** | 70-79점 | 보통 - 개선 권장 |
| **C** | 60-69점 | 부족 - 개선 필요 |
| **D** | 50-59점 | 매우 부족 - 즉시 개선 |
| **F** | 50점 미만 | 실패 - 전면 수정 |

### 평가 항목

1. **글자수** (가중치: 높음)
   - 3000자 이상: 우수
   - 2000-3000자: 양호
   - 1000-2000자: 부족 ⚠️
   - 1000자 미만: 매우 부족 ⚠️⚠️

2. **키워드 반복** (가중치: 중간)
   - 5-8회: 최적 ✅
   - 3-5회: 양호
   - 10회 이상: 스팸 위험 ⚠️
   - 0-3회: 부족 ⚠️

3. **해시태그** (가중치: 중간)
   - 8-10개: 최적 ✅
   - 5-8개: 양호
   - 15개 이상: 스팸 위험 ⚠️
   - 5개 미만: 부족 ⚠️

## 💡 최적화 원리

### 1. 금칙어 치환

금칙어 파일에서 금칙어와 대체어를 읽어서 자동 치환합니다.

**예시:**
- "효과" → "약효"
- "병원" → "클리닉" (랜덤 선택)
- "네요" → "습니다" (랜덤 선택)
- "하더라구요" → (삭제)

### 2. AI 패턴 다양화

반복적인 AI 문장 패턴을 자연스러운 표현으로 변경합니다.

**예시:**
- "정말 고민이 많습니다" → "어떻게 해야 할지 모르겠어요"
- "절로 나오고" → "저도 모르게 나오고"
- "고생하고 있는" → "불편함을 느끼는"

### 3. 키워드 밀도 최적화

원고에서 키워드 출현 횟수를 5-8회로 맞춥니다.

**방법:**
- "이런", "이거", "그런" 등의 대명사를 키워드로 자연스럽게 교체

### 4. 해시태그 생성

키워드와 브랜드를 기반으로 8-10개의 해시태그를 자동 생성합니다.

**예시:**
```
#팔꿈치 쿠션 보호대 #팔꿈치 #쿠션 #보호대 #초록입환 #건강관리 #관절건강 #후기
```

### 5. 제목 생성 (제목이 없는 경우)

SEO 최적화된 제목을 자동 생성합니다 (15-40자).

**템플릿 예시:**
- "{키워드} 추천 정보 (후기 모음)"
- "{키워드} 어떤 게 좋을까요?"
- "{키워드} 정보 공유"

## 🔧 고급 사용법

### 코드에서 직접 사용

```python
from blog_optimizer import BlogOptimizer

# 초기화
optimizer = BlogOptimizer()

# 단일 텍스트 최적화
result = optimizer.optimize_text(
    text="원고 내용...",
    keyword="팔꿈치 쿠션 보호대",
    brand="초록입환",
    title=""  # 비어있으면 자동 생성
)

print(result['optimized_text'])
print(result['optimized_title'])
print(result['hashtags'])
print(result['c_rank_check'])

# 엑셀 파일 최적화
result = optimizer.optimize_excel(
    input_file='작업 의뢰용 데이터.xlsx',
    output_file='결과.xlsx'  # 옵션
)
```

### 금칙어 파일 커스터마이징

1. `금칙어 수정사항 모음.txt` 파일 편집
2. 한 줄에 금칙어, 다음 줄에 대체어
3. 빈 줄로 구분
4. "/" 또는 ","로 여러 대체어 제공 시 랜덤 선택

**예시:**
```
정말
진짜 / 확실히 / 분명히

너무
엄청 / 많이 / 굉장히

효과
약효

하더라구요
(삭제)
```

## ⚠️ 주의사항

1. **글자수 부족**: 원고가 너무 짧으면 (1000자 미만) C랭크 점수가 낮게 나옵니다. 최소 2000자 이상 권장.
2. **키워드 과다**: 키워드를 너무 많이 반복하면 스팸으로 인식될 수 있습니다.
3. **AI 느낌**: 시스템이 AI 패턴을 어느 정도 제거하지만, 완전히 사람처럼 쓰기는 어렵습니다. 최종 검토를 권장합니다.
4. **금칙어 정확도**: 금칙어 파일에 정확한 대체어를 입력해야 자연스러운 결과가 나옵니다.
