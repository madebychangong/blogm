# 🤖 웹 AI로 블로그 원고 최적화하기

Python 환경 없이 ChatGPT, Claude 등 웹 AI로 블로그 원고를 최적화하는 방법

---

## 🚀 빠른 시작 (3단계)

### 1단계: AI에게 프롬프트 전달

`AI_프롬프트_전체.txt` 파일을 열어서 **전체 내용을 복사**한 후, ChatGPT나 Claude에 붙여넣기

### 2단계: 원고 제공

다음 형식으로 원고와 정보를 입력:

```
키워드: 팔꿈치 쿠션 보호대
브랜드: 초록입환
목표 글자수: 2000

원고:
요즘 팔꿈치가 아파서 고민이에요. 쿠션 보호대를 찾아보고 있는데...
(원고 내용)
```

### 3단계: 결과 받기

AI가 다음을 제공합니다:
- ✅ SEO 분석 결과
- ✅ 최적화된 원고 (2000자+)
- ✅ 해시태그 10개
- ✅ SEO 제목
- ✅ C랭크 점수

---

## 💡 사용 예시

### 예시 1: 짧은 원고 확장

**입력:**
```
키워드: 초록입환
브랜드: 초록입환
목표 글자수: 2000

원고:
초록입환 먹어본 사람들 후기 궁금해요.
효과 있나요? 부작용은 없나요?
(총 500자 정도의 짧은 글)
```

**출력:**
- 2133자로 확장
- S등급 (95점)
- 해시태그 10개 포함
- 금칙어 모두 치환
- 자연스러운 문체

### 예시 2: 금칙어만 치환

**입력:**
```
키워드: 피부과 치료
목표 글자수: 유지

원고:
병원에서 진단받고 효과 좋은 치료 시술을 받았어요.
의사 선생님이 주의사항을 알려주셨는데...
(이미 긴 원고)
```

**출력:**
- 글자수 유지
- 금칙어만 치환:
  - "병원" → "센터"
  - "진단" → "확인"
  - "효과" → "약효"
  - "시술" → (삭제)
  - "의사" → "담당의"
  - "주의사항" → "주의할 점"

## 📋 사용 팁

### 1. 첫 대화에서 프롬프트 전달

대화 시작할 때 `AI_프롬프트_전체.txt` 전체를 먼저 복붙하세요.

### 2. 원고는 두 번째 메시지로

프롬프트 입력 후, 다음 메시지로 원고 입력:

```
키워드: [키워드]
브랜드: [브랜드명]
목표 글자수: 2000

원고:
[원고 내용...]
```

### 3. 부족한 부분 추가 요청

결과가 만족스럽지 않으면:

```
- "키워드를 2회 더 추가해줘"
- "글자수를 2500자로 늘려줘"
- "해시태그 5개 더 만들어줘"
- "좀 더 자연스럽게 수정해줘"
```

### 4. 여러 원고 일괄 처리

한 번에 여러 개 요청 가능:

```
다음 3개 원고를 각각 최적화해줘:

[원고1]
키워드: XXX
원고: ...

[원고2]
키워드: YYY
원고: ...

[원고3]
키워드: ZZZ
원고: ...
```

## ❓ FAQ

### Q1: 프롬프트가 너무 길어요

**A1:** 괜찮습니다. Claude는 긴 프롬프트를 잘 처리합니다. ChatGPT는 가끔 짤릴 수 있으니 Claude 추천.

### Q2: 매번 프롬프트를 입력해야 하나요?

**A2:** 같은 대화 세션에서는 한 번만 입력하면 됩니다. 여러 원고를 계속 최적화할 수 있어요.

### Q3: 결과가 마음에 안 들면?

**A3:** "좀 더 자연스럽게 수정해줘" 같이 추가 요청하세요. AI가 재작업합니다.

### Q4: 금칙어가 제대로 안 치환돼요

**A4:** "금칙어 목록을 다시 확인하고 모두 치환해줘"라고 요청하세요.

### Q5: 글이 너무 AI 같아요

**A5:** "더 자연스럽고 구어체로, AI 느낌을 완전히 없애줘"라고 요청하세요.

### Q6: 키워드가 억지로 들어간 느낌

**A6:** "키워드를 더 자연스럽게 배치해줘, 억지로 넣은 느낌 빼고"라고 요청.

## 🎯 실전 활용법

### 1. 아침 작업 루틴

```
1. ChatGPT/Claude 열기
2. 프롬프트 붙여넣기
3. 오늘 작성할 원고 3-5개 준비
4. 하나씩 최적화 요청
5. 결과 복사해서 블로그 예약 게시
```

### 2. 긴급 게시물

```
1. Claude 열기 (빠름)
2. 프롬프트 붙여넣기
3. 원고 입력
4. 1분 대기
5. 바로 복사해서 게시
```

### 3. 일괄 처리

```
1. 원고 10개 준비 (메모장)
2. ChatGPT에 프롬프트 입력
3. "10개 원고를 각각 최적화해줘" + 원고들 붙여넣기
4. 30분 후 모든 결과 받기
5. 일주일치 예약 게시
```
