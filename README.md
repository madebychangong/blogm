# 네이버 블로그 분석기 - 웹 버전

2025 네이버 C-Rank 기준으로 블로그를 분석하는 웹 애플리케이션입니다.

## 🎯 기능

### 무료 버전
- ✅ 게시글별 SEO 점수 분석
- ✅ 콘텐츠 품질 점수 분석 (글자수, 이미지, 동영상, 문단구조)
- ✅ 블로그 전체 랭크 (S~F 등급)
- ✅ 예상 유입 랭크 (SEO 기반)

### 멤버십 전용 (추후 구현)
- ❌ 포스팅 빈도 및 주기 분석
- ❌ 조회수/공감/댓글 통계
- ❌ 경쟁 블로그 비교
- ❌ 상세한 개선 방안 제안

## 📊 점수 체계

### 개별 게시글 (100점)
**SEO 점수 (100점, 감점식)**
- 제목 길이 (15~40자 권장)
- 해시태그 개수 (8~10개 권장)
- 키워드 밀도
- 링크 활용

**콘텐츠 품질 (100점)**
- 글자수 (45점): 3000자 이상 권장
- 이미지 (35점): 10장 이상 권장
- 동영상 (10점): 1개 이상 권장
- 문단 구조 (10점): 5개 이상 권장

### 블로그 랭크 (S~F)
- 기본: 평균 게시글 점수
- 보너스: 포스팅 일관성, 활동성

### 예상 유입 랭크 (S~F)
- SEO 점수 평균만 반영

## 🚀 설치 및 실행

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 서버 실행
```bash
python app.py
```

### 3. 브라우저 접속
```
http://localhost:8000
```

## 📁 파일 구조

```
blog-analyzer-web/
├── app.py              # FastAPI 메인 서버
├── analyzer.py         # 블로그 크롤링 및 분석 로직
├── requirements.txt    # 필요한 패키지 목록
├── README.md          # 이 파일
└── static/
    ├── index.html     # 메인 페이지
    ├── style.css      # 스타일시트 (순수 CSS)
    └── script.js      # JavaScript 로직
```

## 📝 사용 방법

1. 블로그 ID 입력 (예: coco_hodu_)
2. "분석 시작" 버튼 클릭
3. 결과 확인

## ⚠️ 제한사항

### 무료 버전
- 최대 10개 게시글만 분석
- 포스팅 빈도 분석 없음
- 참여도 통계 없음

### 크롤링
- 네이버 블로그 구조 변경 시 작동 안 될 수 있음
- 비공개 블로그는 분석 불가

## 🔧 기술 스택

- **Backend**: FastAPI, Python 3.8+
- **Frontend**: HTML, CSS, JavaScript (순수)
- **크롤링**: requests, BeautifulSoup4
- **서버**: Uvicorn
