==============================================
📝 웹 AI에 복붙할 사용 예시
==============================================

# 1단계: 아래 프롬프트를 ChatGPT/Claude에 복붙
---------------------------------------------

당신은 네이버 블로그 SEO 전문가입니다. 제가 제공하는 블로그 원고를 다음 기준에 맞게 최적화해주세요:

**최적화 기준:**
- 글자수: 2000자 이상
- 통키워드: 5-8회 반복
- 첫 문단 키워드: 2회 (2문장 이상 간격)
- 키워드로 시작하는 문장: 4개 이상
- 서브키워드: 18개 이상 (2회 이상 등장 단어)
- 해시태그: 8-10개
- C랭크 목표: S등급 (85점 이상)

**금칙어 절대 금지 (반드시 대체):**
- 병원 → 센터, 클리닉
- 진단 → 확인, 판정
- 효과 → 약효
- 시술 → (삭제) 또는 치료
- 의사 → 담당의, 선생님
- 주의사항 → 주의할 점
- 증상 → 증.상
- 환자 → 환.자
- 재발 → 다시 그러고, 제자리
- 네요 → 고, 어요, 습니다

**작업 순서:**
1. 금칙어를 대체어로 치환
2. AI 특유 표현을 자연스럽게 변경
3. 글자수가 2000자 미만이면 자연스럽게 확장
4. 키워드를 5-8회 자연스럽게 삽입
5. 해시태그 8-10개 생성
6. SEO 제목 생성

**출력 형식:**
```
📊 SEO 분석
- 글자수: X자
- 통키워드: X회
- C랭크: X점 (등급)

🏷️ 해시태그
[해시태그 10개]

📌 제목
[SEO 제목]

📝 최적화된 원고
[전체 원고...]
```

준비됐으면 "준비 완료"라고 답해주세요.


# 2단계: AI가 "준비 완료" 하면 아래 형식으로 원고 입력
---------------------------------------------

키워드: 팔꿈치 쿠션 보호대
브랜드: 초록입환
목표 글자수: 2000

원고:
팔꿈치 쿠션 보호대 정보 찾고 있어요.
요즘 팔꿈치가 자꾸 아프고 불편해서 보호대를 알아보고 있는데요.
어떤 제품이 좋은지 잘 모르겠더라고요.

주변 사람들한테 물어보니 쿠션이 들어간 보호대가 좋다고 하던데
실제로 효과가 있는지 궁금합니다.

사용해본 분들 계신가요?
후기 좀 알려주세요.


# 3단계: 결과 받고 확인사항
---------------------------------------------

✅ 체크리스트:
- [ ] 글자수 2000자 이상인가?
- [ ] 금칙어가 모두 치환됐는가?
- [ ] 키워드가 5-8회 등장하는가?
- [ ] 자연스러운 문체인가?
- [ ] 해시태그 8-10개 생성됐는가?
- [ ] C랭크 점수가 85점 이상인가?


# 4단계: 추가 요청 예시
---------------------------------------------

만족스럽지 않으면 다음과 같이 요청:

"키워드를 2회 더 추가해줘"
"좀 더 자연스럽게 수정해줘"
"글자수를 2500자로 늘려줘"
"해시태그 5개 더 만들어줘"
"금칙어가 남아있는지 재확인해줘"


==============================================
💡 팁
==============================================

1. 첫 대화에서 프롬프트(1단계)를 한 번만 입력하면
   같은 세션에서 계속 사용 가능

2. 여러 원고를 한 번에 처리하려면:
   "다음 3개 원고를 각각 최적화해줘"
   [원고1]
   [원고2]
   [원고3]

3. 금칙어만 치환하고 싶으면:
   "목표 글자수: 유지 (금칙어만 치환)"

4. 더 자세한 분석을 원하면:
   "C랭크 점수를 항목별로 상세하게 분석해줘"
