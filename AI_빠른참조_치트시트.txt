해시태그:         8-10개
C랭크 목표:       S등급 (85점 이상)

## 💬 AI 표현 → 자연스러운 표현
================================================================================

정말 고민이 많습니다        → 정말 고민되네요, 어떻게 해야 할지 모르겠어요
절로 나오                   → 자연스럽게 나오, 저도 모르게 나오
고생하고 있는               → 힘들어하는, 어려움을 겪는
이렇게 글을 올려봅니다      → 여쭤보고 싶어서요, 조언 구하러 왔어요
솔직히                      → 사실, 실제로
정말                        → 진짜, 확실히
너무                        → 엄청, 많이, 굉장히

## 📝 글 확장 템플릿 (복붙용)
================================================================================

경험담:
저도 비슷한 고민을 오랫동안 해왔는데요, 주변 분들도 많이 겪는 문제더라구요.
특히 나이가 들수록 이런 문제가 더 심해지는 것 같아서 걱정이에요.

정보공유:
혹시 같은 고민을 하시는 분들을 위해 제가 알아본 정보를 공유해드릴게요.
완벽한 정보는 아니지만, 조금이라도 도움이 되었으면 좋겠습니다.

고민:
이런 상황이 계속되니까 일상생활이 너무 불편하더라고요.
간단한 일상 동작 하나하나가 다 신경 쓰이고, 자꾸 의식하게 되니까 스트레스도 쌓이고요.

조언요청:
혹시 이런 경험 있으신 분 계신가요? 어떻게 해결하셨는지 정말 궁금해요.
작은 팁이라도 공유해주시면 큰 도움이 될 것 같습니다.

## 🎯 키워드 삽입 예시
================================================================================

키워드: 팔꿈치 쿠션 보호대

문장 시작:
✅ 팔꿈치 쿠션 보호대 관련해서 정보 찾아봤어요.
✅ 팔꿈치 쿠션 보호대를 사용하면서 느낀 점을 공유합니다.
✅ 팔꿈치 쿠션 보호대 선택 시 고려할 점이 많더라고요.
✅ 팔꿈치 쿠션 보호대 제품별 차이점도 알아봤습니다.

문장 중간:
✅ 제가 사용한 팔꿈치 쿠션 보호대 후기 남겨요.
✅ 요즘 팔꿈치 쿠션 보호대 찾는 분들 많더라고요.

조각 활용:
✅ 팔꿈치 관리 방법, 쿠션 재질, 보호대 착용법

## 🏷️ 해시태그 생성 규칙
================================================================================

1. 통키워드 포함 (#팔꿈치 쿠션 보호대)
2. 조각키워드 각각 (#팔꿈치 #쿠션 #보호대)
3. 브랜드명 (#초록입환)
4. 카테고리 (#건강정보 #일상)
5. 관련 키워드 (#제품후기 #사용법)

총 8-10개, 띄어쓰기 포함 가능

예시:
#팔꿈치 쿠션 보호대 #팔꿈치 #쿠션 #보호대 #초록입환 #건강정보 #일상 #제품후기

## 🚀 AI 플랫폼별 특징

+ 무료 플랜 있음

Claude (claude.ai)
+ 긴 프롬프트 잘 이해
+ 한국어 자연스러움
- 속도 약간 느림

Gemini (gemini.google.com)
+ 무료, 한국어 지원
- 규칙 준수율 낮음

## 💡 시간 절약 팁
================================================================================

1. 프롬프트 한 번만 입력
   → 같은 대화에서 여러 원고 처리

2. 템플릿 메모장에 저장
   → 복붙만 하면 됨

3. 일괄 처리
   "다음 5개 원고를 각각 최적화해줘"

4. 자동 검증 요청
   "금칙어 재확인하고 점수 재계산해줘"

## 📞 트러블슈팅
================================================================================

금칙어가 안 치환됨
→ "금칙어 목록 재확인하고 모두 치환해줘"

키워드가 부족함
→ "키워드를 X회 더 자연스럽게 추가해줘"

글이 너무 AI 같음
→ "더 자연스럽고 구어체로, AI 느낌 완전히 제거해줘"

글자수가 부족함
→ "경험담과 조언요청 추가해서 2500자로 늘려줘"

C랭크 점수가 낮음
→ "어떤 부분이 부족한지 분석하고 개선해줘"
