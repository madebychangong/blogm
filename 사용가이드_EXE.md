# 📱 블로그 SEO 최적화 시스템 사용 가이드

간단한 GUI로 블로그 원고를 네이버 C랭크 기준에 맞게 자동 최적화!

---

## 🚀 빠른 시작

### 1. 프로그램 실행

`블로그SEO최적화.exe` 더블클릭

### 2. 파일 선택

- **"📁 파일 선택"** 버튼 클릭
- Excel 파일 (.xlsx) 또는 TXT 파일 (.txt) 선택

### 3. 설정 (선택사항)

- **키워드:** TXT 파일은 자동 추출, 수동 입력 가능
- **브랜드명:** 제품 브랜드 입력 (선택)
- **목표 글자수:** 기본 2000자 (필요시 변경)

### 4. 최적화 실행

**"🚀 최적화 시작"** 버튼 클릭

### 5. 결과 확인

- 진행 상황이 로그 창에 표시됩니다
- 완료 후 **"📂 결과 폴더 열기"** 클릭

---

## 📂 파일 형식

### Excel 파일 (.xlsx)

**필수 컬럼:**
- 날짜
- 브랜드
- 키워드
- 원고
- 글자수(공백포함)
- 통키워드 반복수
- 조각키워드 반복수
- 서브키워드 목록 수

**결과:** 같은 폴더에 `원본파일_고급최적화.xlsx` 생성

### TXT 파일 (.txt)

**형식:**
```
# 키워드 관련 제목

원고 내용...
```

**결과:** 같은 폴더에 `원본파일_최적화.txt` 생성

## 🎯 최적화 기준

### C랭크 기준

1. ✅ **글자수:** 2000자 이상
2. ✅ **통키워드:** 5-8회 출현
3. ✅ **조각키워드:** 각각 충분히 출현
4. ✅ **서브키워드:** 18개 이상
5. ✅ **첫 문단:** 키워드 2회 (2문장 이상 간격)
6. ✅ **키워드 시작 문장:** 4개 이상
7. ✅ **해시태그:** 8-10개

### 등급 체계

| 등급 | 점수 | 설명 |
|------|------|------|
| S | 95-100 | 완벽 |
| A | 85-94 | 우수 |
| B | 75-84 | 양호 |
| C | 65-74 | 보통 |
| D | 55-64 | 미흡 |
| F | 0-54 | 불합격 |

--------------------------------------------------------------------------------

✅ 키워드 반복 양호 (5-8회)
✅ 해시태그 적정 (8-10개)

📌 제목
--------------------------------------------------------------------------------

**추가되는 컬럼:**
- 추천_해시태그
- C랭크_점수
- C랭크_등급
- C랭크_제안사항
- 최적화_변경사항

## 🔧 고급 사용법

### 여러 파일 일괄 처리 (Excel)

Excel 파일에 여러 행의 원고를 입력하면 자동으로 모두 처리됩니다!

### 목표 설정 변경

- **짧은 글 (1500자):** 목표 글자수를 1500으로 설정
- **긴 글 (3000자):** 목표 글자수를 3000으로 설정

### 금칙어 커스터마이징

`금칙어 수정사항 모음.txt` 파일을 메모장으로 열어 수정:

```
금칙어
대체어1 / 대체어2 / 대체어3

효과
약효

병원
병의원 / 센터 / 클리닉
```

**형식:**
- 금칙어 (첫 줄)
- 대체어들 (둘째 줄, `/` 또는 `,`로 구분)
- 빈 줄
- 다음 금칙어...

## ⚠️ 주의사항

### 필수 파일

**중요:** `금칙어 수정사항 모음.txt` 파일을 실행 파일과 **같은 폴더**에 보관하세요!

```
📁 내폴더/
  ├── 블로그SEO최적화.exe        ← 실행 파일
  └── 금칙어 수정사항 모음.txt    ← 필수!
```

### API 키 설정 (필요한 경우)

프로그램이 Anthropic Claude API를 사용하는 경우:

**Windows:**
```cmd
set ANTHROPIC_API_KEY=your_api_key_here
```

**환경변수로 영구 설정:**
1. 시스템 속성 → 환경 변수
2. 새로 만들기
3. 변수 이름: `ANTHROPIC_API_KEY`
4. 변수 값: `your_api_key_here`

## ❓ FAQ

### Q1: 프로그램이 실행되지 않아요

**A1:** Windows Defender 경고가 뜰 수 있습니다.
- "추가 정보" 클릭 → "실행" 선택

**A2:** Visual C++ 재배포 패키지 필요
- https://aka.ms/vs/17/release/vc_redist.x64.exe 설치

### Q2: 금칙어 파일을 찾을 수 없다고 나와요

**A:** 실행 파일과 같은 폴더에 `금칙어 수정사항 모음.txt`를 복사하세요.

### Q3: Excel 파일이 열리지 않아요

**A:** Excel 파일이 다른 프로그램에서 열려있으면 저장할 수 없습니다. 닫고 다시 시도하세요.

### Q4: 최적화가 너무 오래 걸려요

**A:** 정상입니다. 글자수가 적은 원고일수록 더 많은 내용을 생성해야 해서 시간이 걸립니다.
- 500자 → 2000자: 약 30초~1분
- 1500자 → 2000자: 약 10~20초

### Q5: 결과 글이 너무 AI 같아요

**A:** 금칙어 파일에 AI 특유의 표현들을 추가하고, 여러 번 돌려서 가장 자연스러운 결과를 선택하세요.

### Q6: 키워드가 자동으로 잘 안 잡혀요

**A:** TXT 파일 첫 줄을 `# 키워드 ...` 형식으로 작성하거나, GUI에서 수동으로 입력하세요.

## 🎯 실전 워크플로우

### 1단계: 원고 준비

```
원고1.txt, 원고2.txt, 원고3.txt 작성
```

### 2단계: 최적화

각 파일을 프로그램에서 열어 최적화

### 3단계: 결과 확인

```
원고1_최적화.txt
→ C랭크: S등급 (95점) ✅

원고2_최적화.txt
→ C랭크: B등급 (80점) ⚠️ 재작업

원고3_최적화.txt
→ C랭크: A등급 (90점) ✅
```

### 4단계: 게시

- 최적화된 원고 복사
- 해시태그 복사
- 네이버 블로그에 붙여넣기
